
Bu uygulama, seçtiğiniz bir klasördeki tüm görüntüleri belirtilen boyutlara yeniden boyutlandırır ve yeniden boyutlandırılmış görüntüleri yeni bir klasöre kaydeder.

## Özellikler
- Belirtilen klasördeki tüm `.jpg`, `.jpeg` ve `.png` dosyalarını yeniden boyutlandırır.
- Kullanıcı tarafından girilen genişlik ve yükseklik değerlerine göre yeniden boyutlandırma yapar.
- Yeniden boyutlandırılmış görüntüleri orijinal klasörde `boyutlanmis` adlı yeni bir klasöre kaydeder.
- Kullanıcı dostu grafik arayüzü.
- Görsellerin ortalama boyutunu gösterir.



This application resizes all images in a selected folder to specified dimensions and saves the resized images in a new folder.

## Features
- Resizes all `.jpg`, `.jpeg`, and `.png` files in the specified folder.
- Resizes images based on user-provided width and height values.
- Saves resized images in a new folder named `boyutlanmis` within the original folder.
- User-friendly graphical interface.
- Displays the average size of the images.
